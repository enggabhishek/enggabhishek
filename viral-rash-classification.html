Viral Rash Classification

  1. Home
  2. Viral Rash Classification

Introduction

This project aims to classify three infectious skin diseases - Chickenpox, Measles, and Monkeypox - using various deep learning models. The goal is to compare the performance of different architectures on a relatively small dataset of medical images.

Dataset

  • Total classes: 3 (Chickenpox, Measles, Monkeypox)
  • Training data: 160 images per class (480 total)
  • Validation data: 33 images per class (99 total)

Analysis

Loss

Accuracy

  1. Custom CNN

    • Best Train Accuracy: 73.11% (Epoch 14)

    • Best Validation Accuracy: 56.57% (Epoch 7)

    • Final Validation Loss: 2.0372 (Epoch 14)

  2. Inception ResNet V2

    • Best Train Accuracy: 98.35% (Epoch 12)

    • Best Validation Accuracy: 90.91% (Epoch 11)

    • Final Validation Loss: 0.5338 (Epoch 13)

  3. ResNet50

    • Best Train Accuracy: 99.79% (Epoch 10)

    • Best Validation Accuracy: 90.91% (Epoch 7-8)

    • Final Validation Loss: 0.4050 (Epoch 10)

  4. NesNet-A-Large

    • Best Train Accuracy: 99.58% (Epoch 4)

    • Best Validation Accuracy: 85.86% (Epoch 3)

    • Final Validation Loss: 0.8819 (Epoch 7)

Conclusion

The ResNet50 and Inception ResNet V2 performed the best overall, with ResNet50 slightly outperforming Inception ResNet V2 in terms of both validation accuracy and lower validation loss.

Balanced depth and architecture: ResNet50 uses skip connections (residuals) to avoid vanishing gradient problems in deep networks, helping it generalize better.

Less overfitting: Despite high training accuracy, the validation accuracy remained stable, showing that the model is well-regularized.

Inception ResNet V2 also performs well, but ResNet50’s lower final validation loss and stability make it the top performer here.
